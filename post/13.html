deepin的自动代理设置方法

2018年02月28日

连接好ss-qt后，还需要设置代理才生效，未免每次都去打开手动设置的麻烦，可以用pac文件实现自动代理。 文件可以在github找也可以自己生成，安装genpac就可以自己生成了。 ##安装pip后安装genpac： ```sudo apt install python3-pip``` ```sudo pip3 install genpac``` ##生成pac文件： ```genpac --pac-compress --pac-proxy 'SOCKS5 127.0.0.1:1080' --format pac -o ~/auto.pac``` 这里注意指定的端口与你的ss本地端口一致，我这是1080端口。 ##在deepin的 设置--网络--系统代理--自动 填入pac文件位置： ```file:///home/登录用户名/auto.pac``` ##大功告成，可以愉快的玩耍了。。。 ![deepin](https://bbs.deepin.org/data/attachment/forum/201802/28/154549nnhhtpe1n0mmvvgm.png)
