Linux下无法使用add-apt-repository命令

2018年03月19日

##报错如下: `add-apt-repository command not found` ##解决方案： ```shell sudo apt-get install python-software-properties sudo apt-get update ``` ##最后在执行： ```shell sudo apt install software-properties-common sudo apt-get update ``` ##Done.
